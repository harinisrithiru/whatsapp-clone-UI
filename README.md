# WhatsApp Clone UI Design
A UI/UX design practice project inspired by the WhatsApp onboarding flow, created using Figma.
📌 **Project Overview**
This project focuses on designing the initial user onboarding experience with a clean and simple interface.
The screens designs included are:
- Welcome Screen
- Language Selection Screen
- Phone Number Entry Screen
## 🎯 Objective
The goal of this project was to understand user flow, screen design, and interactive prototyping while practicing UI/UX principles.
## 🛠️ Tools Used
- Figma
- Iconify (for icons)
- Figma Community resources
## 📚 Learning Outcomes
Through this project, I learned:
- Creating wireframes and UI layouts
- Building interactive prototypes in Figma
- Using icons and design resources effectively
- Importance of spacing, alignment, and consistency in UI design
